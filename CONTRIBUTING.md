# Contributing guide

## Requirements

- [pnpm](https://pnpm.io/)

## Setup

Install libraries

```sh
pnpm i
```

Start it and open the url mentioned (http://localhost:8080 by default except if port is already in use)
```sh
pnpm run start
```

## Submit changes

Check if it builds correctly
```sh
pnpm run build
```

If everything is ok, you can create a Pull Request to submit your changes.
